# Данный скрипт помогает пересоздать услуги, восстановив размеры миниатюр до дефолтных.
Перед дублированием: советую сразу перезагрузить в переносимые услуги все обложки в соотношении сторон 1:1, заранее спроектировав их с осознанием что боковые стороны на миниатюрах обрежутся примерно по 8.25% (с каждой).

---

## 📦 Гайд: как пользоваться скриптом пересоздания услуг VK

---

## Что лежит в архиве

```
vk-services-tool/
├── create_services.py                          ← создаёт копии услуг (с правильным кропом)
├── delete_services.py                          ← удаляет старые услуги
├── Сбор ID услуг - код для консоли браузера.txt  ← JS-код для сбора ID
└── README.txt                                  ← этот гайд
```

---

## Шаг 0. Установи Python (один раз)

1. Перейди на [python.org/downloads](https://www.python.org/downloads/)
2. Нажми **«Download Python 3.x.x»**
3. Запусти установщик

> **⚠️ ОБЯЗАТЕЛЬНО поставь галочку `Add Python to PATH` на первом экране!**

4. Нажми **«Install Now»**, дождись окончания

---

## Шаг 1. Установи зависимости (один раз)

Открой командную строку:
- Нажми `Win + R`
- Набери `cmd`
- Нажми Enter

Введи команду:
```cmd
pip install requests Pillow
```

Если пишет «pip не найден»:
```cmd
python -m pip install requests Pillow
```

---

## Шаг 2. Получи Access Token

1. Перейди на **[vkhost.github.io](https://vkhost.github.io/)**
2. Нажми **«VK Admin»**
3. Нажми **«Разрешить»**
4. Тебя перекинет на страницу с длинной ссылкой в адресной строке
5. Скопируй из адресной строки часть **от `access_token=` до `&expires_in`**

```
Пример адресной строки:
https://oauth.vk.com/blank.html#access_token=vk1.a.ДЛИННЫЙ_НАБОР_СИМВОЛОВ&expires_in=0&user_id=...
                                              ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                              копируешь только эту часть
```

> ⚠️ Токен живёт ограниченное время. Если скрипт выдаёт ошибку авторизации — получи новый.

---

## Шаг 3. Узнай Group ID

1. Открой свою группу VK
2. Посмотри адресную строку:
   - Если `vk.com/club42094082` — ID = **42094082**
   - Если `vk.com/my_short_name` — открой **Управление → Информация** → там будет числовой ID
   - Или зайди на [regvk.com/id](https://regvk.com/id/) → вставь ссылку на группу → получишь ID

---

## Шаг 4. Собери ID услуг

1. Открой свою группу ВК → раздел **«Товары»** или **«Услуги»**
2. Нажми `F12` (откроется панель разработчика)
3. Перейди на вкладку **«Console»** (Консоль)
4. Открой файл **«Сбор ID услуг - код для консоли браузера.txt»** из архива
5. Скопируй весь код оттуда
6. Вставь в консоль браузера и нажми **Enter**
7. Скрипт выведет список ID — **скопируй его**

---

## Шаг 5. Настрой скрипт

Открой файл `create_services.py` в блокноте:
- ПКМ → **Открыть с помощью → Блокнот**
- Или в cmd: `notepad create_services.py`

Найди блок настроек в начале файла и заполни **3 поля**:

```python
# ============ НАСТРОЙКИ ============
ACCESS_TOKEN = "ВСТАВЬ_СЮДА_СВОЙ_ТОКЕН"    ← из шага 2
GROUP_ID     = 42094082                      ← из шага 3
SERVICE_IDS  = [10069128, 10069262, ...]     ← из шага 4
```

### Что куда вставлять:

| Переменная | Что вставить | Откуда |
|---|---|---|
| `ACCESS_TOKEN` | Длинная строка токена в кавычках | Шаг 2 (vkhost.github.io) |
| `GROUP_ID` | Числовой ID группы БЕЗ минуса | Шаг 3 (адресная строка) |
| `SERVICE_IDS` | Список ID в квадратных скобках через запятую | Шаг 4 (консоль браузера) |

**Сохрани файл** (`Ctrl + S`)

Аналогично настрой `delete_services.py` (те же 3 поля).

---

## Шаг 6. Запусти создание новых услуг

```cmd
cd ПУТЬ_К_РАСПАКОВАННОЙ_ПАПКЕ
python create_services.py
```

Например:
```cmd
cd C:\Users\Вася\Downloads\vk-services-tool
python create_services.py
```

> 💡 **Лайфхак**: напиши `cd ` (с пробелом), затем **перетащи папку** из проводника прямо в окно CMD — путь вставится автоматически.

Скрипт:
- Скачает все фото из старых услуг
- Загрузит их заново с правильным кропом
- Создаст новые услуги с теми же названиями, описаниями и ценами
- Сохранит маппинг старых → новых ID в файл `created_services.json`

---

## Шаг 7. Проверь результат

1. Открой группу ВК → Товары/Услуги
2. Убедись что новые услуги выглядят правильно
3. Проверь:
   - ✅ Обложки не обрезаны
   - ✅ Цены на месте
   - ✅ Описания сохранились
   - ✅ Доп. фото на месте

---

## Шаг 8. Удали старые услуги

**Только после проверки!** Это необратимо.

Открой `delete_services.py` в блокноте и убедись что:
- `ACCESS_TOKEN` и `GROUP_ID` заполнены
- `SERVICE_IDS` содержит ID **старых** услуг (те что были до пересоздания)

```cmd
python delete_services.py
```

Скрипт покажет список и попросит подтверждение — введи `YES`.

## ❓ Частые проблемы

### «python не является внутренней командой»
→ Python не добавлен в PATH. Переустанови Python с галочкой **«Add Python to PATH»**

### «No module named requests»
→ Забыл установить зависимости:
```cmd
pip install requests Pillow
```

### «User authorization failed»
→ Токен истёк. Получи новый (Шаг 2)

### «Access denied»
→ Токен не имеет прав на управление товарами. Убедись что нажимал именно **«VK Admin»** на vkhost.github.io

### Скрипт мелькнул и закрылся
→ Не запускай двойным кликом! Открой CMD (`Win+R` → `cmd`) и запусти вручную

### Цена отображается неправильно
→ Убедись что используешь последнюю версию скрипта, где `amount` делится на 100
